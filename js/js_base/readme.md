# js中的数据类型

  基础数据类型：string,number,boolean,undefined,null,symbol

  引用类型:object(对象类型)，function(函数类型),array(数组)，Regexp，Date

  使用typeof来判断数据类型，null不能被正确判断，而且只能判断基础数据类型

  使用instanceof只能判断复杂数据类型
